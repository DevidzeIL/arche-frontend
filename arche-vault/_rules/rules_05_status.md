# Status System

Статус отвечает на вопрос: "Насколько я это понимаю СЕЙЧАС?"

Это личная шкала зрелости понимания, а не оценка качества текста.

---

## Статусы

### seed — идея только зафиксирована
- Мысль появилась, но ещё не проработана
- Минимальное содержание
- Может быть только название и пара ссылок
- **Это нормальное состояние** — большинство заметок начинаются с seed

### growing — начинаю понимать
- Начал разбираться в теме
- Есть базовое содержание
- Появились связи с другими заметками
- Понимание ещё нестабильно, может меняться

### stable — базовое понимание есть
- Есть чёткое понимание темы
- Содержание структурировано
- Связи установлены
- Можно использовать как справочник

### deep — хорошо проработано
- Тема глубоко изучена
- Множественные связи и контекст
- Возможно, есть интерпретации
- Заметка может служить основой для других

### question — есть сомнение или конфликт
- Есть неразрешённые вопросы
- Конфликтующие точки зрения
- Неопределённость в понимании
- **Это ценный статус, а не ошибка** — показывает зоны незнания

---

## Переходы между статусами

- Статус можно и нужно менять
- Можно вернуться назад (stable → growing, если понимание изменилось)
- Нет обязательного пути: seed → growing → stable → deep
- Можно сразу перейти от seed к stable, если тема проста

---

## Критерии выбора статуса

**seed:**
- Только зафиксировал мысль
- Ещё не начал разбираться
- Содержание минимальное

**growing:**
- Начал читать/изучать
- Появились первые связи
- Понимание формируется

**stable:**
- Могу объяснить тему другому
- Есть структурированное понимание
- Связи установлены

**deep:**
- Изучил тему глубоко
- Вижу связи с разными областями
- Могу обсуждать нюансы

**question:**
- Есть явные сомнения
- Конфликт интерпретаций
- Неопределённость в понимании

---

Статус — это не оценка, а навигация по собственному пониманию.
