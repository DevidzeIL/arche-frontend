# Настройка графа в Obsidian

**⚠️ Важно:** Даже с небольшим количеством файлов граф может быть перегружен. Используйте готовые профили из [[ПРОФИЛИ_ГРАФА]] для быстрого старта.

Этот файл объясняет, как настроить граф в Obsidian, чтобы:
- Примеры отображались красиво и были сгруппированы
- Шаблоны не отображались на графе
- Граф оставался читаемым

## Что было сделано

### Для примеров:
- Добавлено поле `group: example` в meta-блок каждого примера
- Добавлен тег `#example` в начало каждого файла

### Для шаблонов:
- Добавлено поле `group: template` в meta-блок каждого шаблона
- Добавлен тег `#template` в начало каждого файла

## Настройка графа

### Способ 1: Фильтрация по тегам (рекомендуется)

1. Откройте граф (Ctrl+G или Cmd+G)
2. В правом верхнем углу нажмите на иконку фильтра
3. В разделе **Tags**:
   - **Показать только:** добавьте `#example` — чтобы видеть только примеры
   - **Скрыть:** добавьте `#template` — чтобы скрыть шаблоны

### Способ 2: Фильтрация по папкам

1. Откройте граф
2. В разделе **Files**:
   - **Показать только:** добавьте папку `Examples/` — чтобы видеть только примеры
   - **Скрыть:** добавьте папку `_templates/` — чтобы скрыть шаблоны

### Способ 3: Комбинированный подход

Используйте оба способа одновременно:
- Показывать: `#example` ИЛИ папка `Examples/`
- Скрывать: `#template` И папка `_templates/`

## Группировка на графе

### По тегам:
- Все заметки с тегом `#example` можно визуально выделить одним цветом
- В настройках графа → **Groups** → выберите группировку по тегам

### По папкам:
- Заметки из папки `Examples/` автоматически группируются
- В настройках графа → **Groups** → выберите группировку по папкам

### По полю group в meta:
- Если используете плагины для работы с frontmatter, можно группировать по полю `group`
- Примеры будут иметь `group: example`
- Шаблоны будут иметь `group: template`

## Цветовая схема

Можно настроить цвета для разных групп:

1. Откройте настройки графа
2. Перейдите в **Groups**
3. Настройте цвета:
   - `#example` → синий или зелёный
   - `#template` → серый (или скрыть)
   - Остальные заметки → по типу (concept, person, work и т.д.)

## Сохранение настроек

После настройки фильтров:
1. Нажмите на иконку сохранения в правом верхнем углу графа
2. Сохраните как "Примеры" или "Без шаблонов"
3. Можно создать несколько сохранённых видов графа

## Полезные советы

- **Используйте поиск на графе** — введите `#example` чтобы найти все примеры
- **Масштабирование** — используйте колесо мыши для увеличения/уменьшения
- **Выделение** — кликните на узел, чтобы увидеть только связанные заметки
- **Экспорт** — можно экспортировать граф как изображение для документации

## Проверка

После настройки проверьте:
- ✅ Примеры видны на графе и сгруппированы
- ✅ Шаблоны не отображаются
- ✅ Связи между примерами видны
- ✅ Граф читаемый и не перегружен
