# XX век (первая половина)

## Общая характеристика
Период с 1900 по 1950 годы — эпоха радикальных изменений: две мировые войны, революции, распад империй, кризис традиционных ценностей. Время, когда старые системы рухнули, а новые ещё не сложились. Это создало почву для экзистенциализма — философии, которая отвечала на вопрос "как жить в мире без гарантий".
